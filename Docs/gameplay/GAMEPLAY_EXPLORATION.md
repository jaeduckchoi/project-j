# 탐험 시스템 기획

## 1. 탐험의 역할

탐험은 식당 운영을 위한 재료 공급 루프다.
플레이어는 지역을 직접 돌아다니며 자원을 모으고, 허브로 돌아와 냉장고에서 보유 재료를 확인하고, `FrontCounter`와 `BackCounter` 조리에 필요한 재료를 준비한다.

## 2. 공용 조작과 상호작용

- 이동 : `WASD` 또는 방향키
- 상호작용 : `E`
- 채집 액션 : 상황에 따라 `Space` 홀드 사용
- 채집 가능 거리에 들어가면 가장 가까운 유효 대상이 자동으로 선택된다.
- 채집 대상이 도구 부족으로 막혀 있어도 차단 사유는 항상 표시한다.

## 3. 시작 도구와 탐험 입문

- 플레이어는 허브에서 나와 `Beach`로 향하는 초반 흐름에서 기본 도구 상자를 확인한다.
- 상자에는 낡은 어망과 낡은 호미가 들어 있으며, 앞으로 탐험과 채집을 통해 식당 재료를 확보하게 된다는 점을 알려 준다.
- 기본 재료 `김치`, `밥`, `고춧가루`, `밀가루`는 허브에서 상시 사용할 수 있고, 그 외 재료는 탐험과 채집으로 확보한다.

## 4. 탐험맵 구조

탐험맵은 `Beach`(바닷가)를 중심 연결 노드로 사용한다.
식당 허브에서 나오면 먼저 바닷가에 도착하고, 모든 탐험은 바닷가에서 퍼지거나 바닷가로 돌아오는 구조를 따른다.

```text
Hub -> Beach -> Sea
             -> DeepForest -> WindHill -> AbandonedMine
             -> Shortcut -> WindHill
```

### Beach / 바닷가

- 크기 : `3840 x 2160`
- 목적 : 허브 외부 출구 직후 첫 화면이자 모든 탐험의 중심 연결 필드
- 연결 : 좌측 어선으로 `Sea`, 우측으로 `DeepForest`, 상단으로 `Shortcut`
- 위험 : 거의 없음. 안전한 입문 환경
- 메모 : 플레이어는 바닷가를 기준으로 식당 복귀와 탐험 확장 흐름을 읽는다.

#### 갯바위 채집

- 바닷가 아래로 이어지는 다리를 따라가면 갯바위 채집 지점이 나온다.
- 상호작용 키가 표시되면 `Space`를 홀드해 채집을 시작한다.
- 캐릭터 위에 채집 게이지가 생성되며, 5초를 유지하면 채집에 성공한다.
- 성공 시 투명 팝업으로 `김`, `바지락`, `홍합` 중 랜덤 3개를 획득한다.

### Sea / 바다

- 크기 : `1920 x 1080`
- 목적 : 바닷가 좌측 어선으로 이동하는 독립 어망 채집 공간
- 연결 : `Beach` 좌측 어선 경로와 왕복
- 위험 : 낮음. 짧은 독립 루프를 우선한다

#### 어망 채집

- 플레이어는 배 위에서만 이동할 수 있다.
- 바라보는 방향으로 `Space`를 홀드하면 어망 투척 거리 가이드가 표시된다.
- 손을 떼면 어망이 바다로 던져지고, 해당 위치에 게이지가 생성된다.
- 15초가 지나면 회수 입력이 다시 활성화된다.
- `Space`를 홀드해 어망을 끝까지 끌어당기면 채집이 완료된다.
- 성공 시 투명 팝업으로 `우럭`, `고등어`, `새우`, `꽃게`, `오징어` 중 랜덤 3개를 획득한다.
- `E` 상호작용으로 "돌아가겠습니까?" 확인 팝업을 열고 `Beach`로 복귀한다.

### DeepForest / 깊은 숲

- 크기 : `2880 x 4320`
- 자원 : `Mushroom`, `Herb`
- 목적 : 재료 다양화와 인벤토리 확장 동기 부여
- 연결 : `Beach` 우측에서 진입하고 `WindHill`로 이어진다
- 구조 : 아래쪽 입구에서 시작해 위쪽으로 깊어지는 세로형 긴 구조
- 위험 : `ForestSwampZone` 감속, 가이드 존, 경로 선택 압박

### WindHill / 바람 언덕

- 크기 : `5760 x 2160`
- 자원 : `Wind Herb`
- 목적 : 반복 파밍과 지름길 개방 이후의 이동 효율 담당
- 연결 : `DeepForest`와 연결되며, 상단에서 `AbandonedMine`으로 이어진다. `Shortcut` 개방 후에는 `Beach`와도 연결된다
- 구조 : 가로로 긴 구조
- 위험 : `WindLaneZone` 돌풍

### Shortcut / 지름길

- 크기 : `1920 x 1080`
- 목적 : `Beach`와 `WindHill`을 잇는 편의 경로
- 해금 : 초반에는 막혀 있으며 골드를 소비해 개방한다
- 연결 : `Beach` 상단과 `WindHill` 사이 왕복
- 위험 : 낮음. 반복 이동 시간을 줄이는 보상 경로

### AbandonedMine / 폐광산

- 크기 : `3840 x 3240`
- 자원 : `Glow Moss`
- 목적 : `WindHill` 상단에서 진입하는 독립 고난도/고보상 지역
- 위험 : `MineDarkness`, 좁은 동선
- 메모 : 보조 위험과 상호작용 오브젝트도 지원 씬 직렬화에 직접 저장된 구성을 기준으로 유지한다.

## 5. 탐험 재료와 식당 연결

- 탐험에서 획득한 재료는 허브의 냉장고 팝업에서 현재 보유 목록으로 확인한다.
- 플레이어는 냉장고에서 확인한 재료를 `FrontCounter`에 올려 조리 묶음을 준비한다.
- 준비된 묶음은 `BackCounter` 기구 조리 입력으로 사용된다.
- 조리 완료 결과물은 다시 `FrontCounter`로 돌아와 최종 완성 단계에 사용된다.
- 탐험은 식당 운영을 위한 공급 루프이며, 식당은 탐험 재료의 소비와 보상 회수 루프다.

## 6. 환경 위험 종류

- **감속 지대** (`MovementModifierZone`)
  늪지나 잔해처럼 지나가기 어려운 지형에 사용한다.
- **어둠 지대** (`DarknessZone`)
  랜턴이 없으면 이동과 시야가 크게 불편해지도록 의도한다.
- **돌풍 지대** (`WindGustZone`)
  일정 주기로 플레이어를 한 방향으로 밀어낸다.

## 7. 포탈 잠금 규칙

- 탐험 지역 진입 포탈은 day/phase 제한을 두지 않는다.
- 허브 복귀 포탈은 언제든 허브로 돌아갈 수 있게 유지한다.
- 포탈은 portal rule 기준의 필요 도구, 필요 평판, 골드 해금 조건을 확인한다.
- 조건이 부족하면 일반 행동 프롬프트 대신 차단 사유를 보여 준다.
- 허브에서 직접 나가는 탐험 진입점은 `Beach`를 우선한다.
- `Shortcut`은 초반에는 막혀 있으며, 골드 소비 조건을 만족하면 `Beach`와 `WindHill`을 연결한다.

## 8. 플레이테스트 확인 항목

- 식당 허브에서 나가면 `Beach`가 첫 화면으로 열리는가?
- `Beach`에서 좌측 어선, 우측 깊은 숲, 상단 지름길 연결이 명확하게 읽히는가?
- 갯바위 채집의 `Space` 홀드 5초 흐름이 명확한가?
- `Sea`의 어망 채집이 다른 지역 채집과 분리된 독립 루프로 동작하는가?
- 탐험 재료가 냉장고 목록과 조리 입력으로 자연스럽게 연결되는가?
- 인벤토리 압박이 의미 있는 수집 판단을 만드는가?
- `Shortcut`이 초반에는 막혀 있고 골드 소비 후 `Beach`와 `WindHill`을 연결하는가?
- `AbandonedMine`이 `WindHill` 상단의 고난도/고보상 지역으로 연결되는가?
