# 탐험 시스템 기획

## 1. 탐험의 역할

탐험은 식당 운영을 위한 재료 공급 루프다.

- 플레이어는 직접 지역을 돌아다니며 재료를 모은다.
- 수급한 재료는 허브 냉장고에서 확인하고, 이후 `CookingUtensils` 조리 입력으로 이어진다.
- 허브 운영 보상은 다시 탐험 동선과 편의 확장에 쓰인다.

## 2. 기본 조작

- 이동: `WASD` 또는 방향키
- 상호작용: `E`
- 채집 액션: 상황에 따라 `Space` 홀드

## 3. 초반 도구와 입문 흐름

- 허브에서 나와 `Beach`로 향하는 초반 동선에서 기본 도구 상자를 만난다.
- 상자에는 낡은 어망, 낡은 호미가 들어 있다.
- 이 연출은 식당 운영만이 아니라 탐험과 채집이 핵심 루프라는 점을 알려 준다.

기본 재료 `김치`, `밥`, `고춧가루`, `밀가루`는 허브에서 상시 사용하고, 그 외 재료는 탐험과 채집으로 확보한다.

## 4. 확장 탐험 축

허브 코어 이후 확장할 탐험 목표 축은 아래와 같다.

```text
Hub -> Beach -> Sea
```

크기와 화면 연출 기준은 gameplay 문서에서 직접 고정하지 않는다.
현재 `Beach`의 화면 기준 정본은 `Docs/scene/BEACH_WHITEBOX.md`를 따르고, `Sea`의 크기 기준은 후속 문서 분리 범위로 둔다.
이번 허브 코어 정합화 범위에서는 `Beach` 이탈 차단과 냉장고 연계만 현재 런타임 규칙으로 보고, `Sea`와 `Space` 홀드 채집은 확장 목표로 남긴다.

### Beach / 바닷가

- 허브 외부 출구 직후의 첫 탐험 허브다.
- 방향 읽기가 쉬워야 한다.
- 집 앞 도구 상자, 갯바위, 부두와 어선, 후속 지역 암시를 담당한다.

#### 갯바위 채집

- 아래쪽 다리를 따라가면 갯바위 채집 지점이 나온다.
- `Space` 홀드 5초 유지 시 채집 성공.
- `김`, `바지락`, `홍합` 중 랜덤 2개를 획득한다.

### Sea / 바다

- `Beach`의 어선으로 이동하는 독립 채집 루프다.
- 배 위에서만 움직일 수 있다.
- 크기와 화면 연출 기준은 현재 후속 문서 분리 범위다.

#### 어망 채집

- 바라보는 방향으로 `Space` 홀드 -> 투척 거리 가이드 표시
- 손을 떼면 어망 투척
- 15초 뒤 회수 가능
- 다시 `Space` 홀드로 회수
- `우럭`, `고등어`, `새우`, `꽃게`, `오징어` 중 랜덤 2개 획득

## 5. 후속 탐험 씬

아래는 현재 코어 범위 밖의 후속 확장 기획이다.

### DeepForest / 깊은 숲

- `Beach` 우측 진입
- 주요 자원: 야채
- 역할: 재료 다양화, 인벤토리 확장 동기
- 위험: 길 읽기 압박

### WindHill / 바람 언덕

- `DeepForest`와 연결
- 주요 자원: 육류
- 역할: 이동 효율과 환경 리듬
- 위험: 돌풍

### Shortcut / 지름길

- `Beach`와 `WindHill` 사이 편의 경로
- 초반 막힘, 골드 소비 후 개방

### AbandonedMine / 폐광산

- `WindHill` 상단의 고위험/고보상 지역
- 주요 자원: 희귀 재료
- 위험: 어둠 지대, 좁은 동선

## 6. 레벨 디자인 관점

- `Beach`는 첫 외부 맵이므로 복잡한 재미보다 방향 읽기가 우선이다.
- 좌측 부두, 아래쪽 갯바위, 우측 후속 진입 방향이 한눈에 읽혀야 한다.
- 탐험은 단순 반복이 아니라 어떤 위험을 감수하고 어떤 재료를 노릴지 고르는 선택이 되어야 한다.
- 메뉴 선정이 탐험 가치에 영향을 준다는 연결이 강할수록 허브와 탐험이 따로 놀지 않는다.
