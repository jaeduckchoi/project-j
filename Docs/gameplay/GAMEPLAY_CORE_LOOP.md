# 게임플레이 코어 루프

## 핵심 흐름

- 허브에서 정리/영업/업그레이드
- 외부 출구로 `Beach`(바닷가) 이동
- 바닷가에서 탐험지 선택
- 지역 이동과 상호작용
- 채집
- 허브 복귀
- 저장/인출
- 레시피 선택과 영업

## 탐험 이동 축

- `Beach`는 식당 허브 외부 출구 직후의 첫 화면이자 중심 연결 노드입니다.
- `Sea`는 바닷가 좌측 어선으로 이동하는 독립 어망 채집 공간입니다.
- `DeepForest`는 바닷가 우측에서 진입하며, 아래쪽 입구에서 위쪽으로 깊어지는 세로형 지역입니다.
- `WindHill`은 깊은 숲과 연결된 가로형 지역이며, 상단에서 `AbandonedMine`으로 이어집니다.
- `Shortcut`은 초반에는 막혀 있고, 골드를 소비해 `Beach`와 `WindHill` 사이를 여는 편의 경로입니다.

## API 연동 기준

- 원격 세션과 스냅샷 반영은 `JongguApiSession`, `GameManager`를 기준으로 유지합니다.
- 안내 문구, 포탈 잠금, 팝업 일시정지 회귀는 현재 런타임 코드 기준으로 직접 확인합니다.
