# 종구의 식당 게임 기획 개요

이 문서는 `Jonggu Restaurant` 프로토타입의 게임 기획서 진입점이다.
세부 시스템 기획은 같은 폴더의 다른 문서에서 다룬다. 코드 경로나 빌더 흐름 같은 구현 기준은 `Docs/project/GAME_DOCS_INDEX.md`를 따라 관련 `Docs/project/*`, `Docs/ui/*`, `Docs/scene/*`, `Docs/build/*` 문서를 참고한다.

## 1. 한 줄 소개

2D 탑다운 탐험으로 재료를 모으고, 식당 허브로 돌아와 작은 식당을 운영하는 1인 프로토타입.

## 2. 핵심 컨셉

- 플레이어는 식당 허브에서 외부 출구로 나가 `Beach`(바닷가)에 도착하고, 바닷가를 중심 연결 노드로 삼아 탐험을 확장한다.
- 플레이어는 탐험 지역에서 재료를 모으고, 식당 허브로 돌아와 메뉴를 정하고 영업을 진행한다.
- 영업 결과는 골드와 평판으로 변환되며, 그 결과가 갈 수 있는 지역과 성장 폭을 정한다.
- 체감 비중은 `탐험 8 : 식당 2`를 목표로 한다. 식당은 짧고 가볍게, 탐험은 길고 결정 위주로 유지한다.

## 3. 디자인 기둥

- **작은 결정의 반복** : 인벤토리 한도와 지역 선택이 반복 플레이의 판단을 만든다.
- **단계적 탐험 확장** : 바닷가를 중심으로 도구, 골드 해금, 위험도 증가가 새로운 지역을 연다.
- **빠른 영업 사이클** : 메뉴 한 번 결정 → 짧은 영업 → 즉시 결과 확인.
- **읽기 쉬운 화면** : 최소한의 UI로 현재 상태와 다음 행동이 한눈에 들어온다.

## 4. 플레이어 경험 흐름

1. **입문** : 식당 허브에서 나와 `Beach`(바닷가)를 첫 탐험 화면이자 중심 연결 필드로 익힌다.
2. **바다** : `Beach` 좌측의 어선 상호작용으로 `Sea`(바다)에 이동해 독립된 어망 채집을 경험한다.
3. **깊은 숲** : `Beach` 우측에서 `DeepForest`(깊은 숲)로 진입하고, 아래쪽 입구에서 위쪽으로 깊어지는 세로형 동선을 익힌다.
4. **바람 언덕** : `DeepForest`에서 `WindHill`(바람 언덕)로 이어지는 넓은 가로형 지역을 탐험한다.
5. **지름길** : 골드를 소비해 `Shortcut`(지름길)을 열고 `Beach`와 `WindHill` 사이 반복 이동 효율을 높인다.
6. **폐광산** : `WindHill` 상단에서 `AbandonedMine`(폐광산)으로 들어가 고난도/고보상 탐험을 진행한다.

## 5. 프로토타입 범위

### 포함

- `WASD` 또는 방향키 이동, `E` 상호작용
- 1개 식당 허브 + 6개 탐험맵 씬
- 재료 전용 인벤토리 슬롯 (`8 → 12 → 16` 확장)
- 도구는 인벤토리 슬롯을 차지하지 않는 영구 해금 아이템
- 단순한 영업 (`serviceCapacity = 3`)
- 데이터 기반 업그레이드 (`골드 + 특정 재료`)
- 허브 전용 창고 보관 시스템

### 제외

- 클릭 이동, 장비 교체 UI, 재료 버리기 시스템
- 복잡한 실시간 식당 운영
- 멀티플레이, 세이브 슬롯, 다국어 UI

## 6. 플레이 가능 씬

| 씬 | 표시 이름 | 크기 | 역할 |
| --- | --- | --- | --- |
| `Hub` | 식당 허브 | `1920 × 1080` | 메뉴/창고/업그레이드/외부 출구 |
| `Beach` | 바닷가 | `3840 × 2160` | 허브 외부 출구 직후 첫 화면, 중심 연결 필드 |
| `Sea` | 바다 | `1920 × 1080` | 바닷가 좌측 어선으로 이동하는 독립 어망 채집 공간 |
| `DeepForest` | 깊은 숲 | `2880 × 4320` | 바닷가 우측 연결, 아래쪽 입구에서 위쪽으로 깊어지는 세로형 지역 |
| `WindHill` | 바람 언덕 | `5760 × 2160` | 깊은 숲과 연결되는 가로형 지역, 폐광산으로 이어지는 간접 연결 지점 |
| `Shortcut` | 지름길 | `1920 × 1080` | 바닷가와 바람 언덕을 잇는 골드 소비 개방 지역 |
| `AbandonedMine` | 폐광산 | `3840 × 3240` | 바람 언덕 상단의 독립 고난도/고보상 지역 |
