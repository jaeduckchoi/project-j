# GeneratedSources 폴더 안내

`Assets/Design/GeneratedSources`는 generated 결과물과 연결되는 디자인 소스를 분리해 두는 위치다.
현재 구조는 아래 기준으로 맞췄다.

- `UI`
  generated UI 작업에 대응되는 버튼, 메시지 박스, 패널 원본
- `Sprites`
  앞으로 generated 월드 스프라이트 원본을 모을 자리
- `Fonts`
  generated 폰트 제작에 대응되는 디자인 참고본이나 시안 자리
- `Data`
  generated 데이터 시트, 표, 기획 원본을 모을 자리

현재 실제 파일은 `UI` 하위에 먼저 정리되어 있고, 나머지 폴더는 앞으로 대응 원본이 들어오면 같은 기준으로 채우면 된다.
이 폴더 아래 디자인 소스 자산 파일명은 기본적으로 kebab-case 기준으로 맞춘다.
