# 로컬 규칙 메모

이 폴더는 개인 PC 전용 AI 규칙과 메모를 두는 자리다.

기본 원칙:

- 공유 저장소에는 이 `README.md`만 유지한다.
- 개인 실행 경로, Unity 설치 경로, 배치 명령, 임시 체크리스트 같은 환경 의존 메모는 각자 로컬에서만 관리한다.
- 여러 사람이 함께 알아야 하는 기준으로 승격된 내용은 `.aiassistant/rules/project` 또는 관련 카테고리 문서, `AGENTS.md`, `CLAUDE.md`에 함께 반영한다.
- 개인 계정 정보, 토큰, 절대 경로처럼 민감하거나 환경 종속적인 값은 공유 규칙 문서로 옮기지 않는다.
